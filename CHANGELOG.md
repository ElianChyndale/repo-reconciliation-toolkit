# Changelog

## 0.1.0

- Added read-only Git and filesystem repository snapshots.
- Added dependency, migration, schema, runtime, environment-name, and optional test
  status collection.
- Added path and remote credential redaction.
- Added two-way and three-way divergence analysis.
- Added Python, JavaScript/TypeScript, and Solidity symbol fingerprints.
- Added required snapshot, dependency, divergence, migration, and backup reports.
- Added synthetic base/left/right fixtures and deterministic machine-readable results.
